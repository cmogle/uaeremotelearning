The Get help now tab should open up to a page that has all buttons visible. The student can then select the most appropriate button for thieir current need. Once they select the button, they will then be walked through the resources (’answers') one by one. When they finish they return back to the get help now page with all the buttons.
The journey needs to feel unambiguously clear and progressive.
This website should feel appealing and familiar to students. They should to feel inclined to explore all pages, but if they have an urgent need, that should be served first. 
Users ages will vary 11-16, boys and girls. >15% of users have an identified ADHD or Dyslexia dignosis.
Dual coding helps maintain student interest and engagement.
Emojis and icons might interest them. 
I think having options of different layouts or skins might appeal to them - particularly if it offers them the utility of self customisation. 
I do not anticipate requiring user registration or login at this time. If customisation can be supported via cookies or similar, recommend that. Alternatively, make the alternate design/skin selection so effortless that it is not felt to be a burden to select mode on each visit.
